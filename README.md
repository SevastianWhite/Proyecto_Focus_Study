## Cómo ejecutar el proyecto

1. Instalar dependencias:
   ```
   pip install -r requirements.txt
   ```

2. Ejecutar Flask desde la carpeta raíz del proyecto:
   ```
   flask run
   ```

3. Abrir en el navegador: `http://127.0.0.1:5000`
